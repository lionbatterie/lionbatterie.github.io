# li-on.github.io
LI/ON est un projet industriel dédié à la réparation, la régénération et le reconditionnement de batteries professionnelles. Il vise à prolonger la durée de vie des batteries existantes, réduire les coûts d’exploitation et limiter l’impact environnemental, dans une logique d’économie circulaire, de traçabilité et de sécurité.
